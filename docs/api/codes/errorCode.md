# API 錯誤碼

## 系統

| 錯誤碼 | 說明 |
|:-:|:-:|
| 0 | 處理成功 |
| 1 | API 設置錯誤 |
| 3 | 系統錯誤 |
| 4 | 資料庫請求失敗 |
| 26 | 參數格式錯誤 或 缺少參數 |
| 27 | 資料驗證失敗 |
| 28 | 處理失敗 |
| 503 | 系統維護中 |
| 999 | 未知錯誤 |

| 2001 | 玩家不存在 |

| 4001 | 玩家仍在競賽中 |

| 4008 | 角色排名不符 |
| 4009 | 完成競賽處理失敗 |
